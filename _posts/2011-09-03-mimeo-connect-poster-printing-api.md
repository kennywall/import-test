{% include JB/setup %}<p>Weve added a newAPI that provides poster printing servicesfor developers.It enables you to price and order commercially printed posters in different sizes, paper, lamination and mounting from within your web and mobile applications.These are the different settings currently available for posters:

	size- 8.5×11, 11×17, 12×18, 18×24, 24×36 or 36×48
	paper- Premium Photo, Heavy Matte or Standard Matte
	lamination- None or Laminated
	mounting- None or Foam Core Mount

We provide five methods currently:

	Proof (POST)
	Proof (GET)
	Quote (GET)
	Shipping Options (GET)
	Order Quote (GET)
	Place Order (POST)
	Order Status (GET)

You will also find common error codes, PHP code samples.I will be adding more code samples as they become available.</p>
