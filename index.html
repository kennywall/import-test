My name is Kin Lane and I am a technology professional with an obsession for Application Programming Interfaces, also know as APIs.

You will find me half the year, traveling the world evangelizing the importance of API literacy, and the other half of the time, working virtually to better understand the world of APIs, and publish my ongoing research, and stories here on my network.

You can read more about me on my about page, if you want to get quickly get up to speed on what I'm up to, here are the top projects, I'm working on:

  • API Evangelist - Where I blog about the business of APIs (primary blog).
  • API Voice - Where I blog about the politics of APIs, how this is more than just tech.
  • API Commons - Mine, and 3Scale's open format to make sure APIs stay openly licensed.
  • APIs.json - Mine, and 3Scale's open format to make sure APIs are discoverable.
  • APIStrat - An API focused conference that I put on in partnership with 3Scale.

You can connect with me via my top social networks, where you will find the real-time stream of what I'm up to:

  • Twitter - My primary social network, where you will find me engaging with the community.
  • Github - Where I publish all of my research, code and stories.
  • Facebook - A more personal network for me, but I'm slowly opening it up to public.
  • LinkedIn - My professional social network, and business persona.
  • Google Plus - While my Google account is prominent in my online world, Googe Plus plays a minor role.
